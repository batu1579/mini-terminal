# `History` 模块

## `history forward[ times]`

-   功能： 向命令行中填充历史记录指针前执行的语句（此操作不会被记录）
-   别名： `up`
-   参数：
    -   `times` ：填充历史记录指针前第几条语句（默认为1）

## `history backward[ times]`

-   功能： 向命令行中填充历史记录指针后执行的语句（此操作不会被记录）
-   别名： `down`
-   参数：
    -   `times` ： 填充历史记录指针后第几条语句（默认为1）

## `history list`

-   功能： 输出历史记录中全部的语句

## `history remove`

-   功能： 删除当前历史记录指针指向的语句

## `history clear`

-   功能： 清空历史记录中的全部语句
