# `Terminal` 模块

## `terminal clear`

-   功能： 用来清空屏幕
-   别名： `clear`

## `terminal exit`

-   功能： 用来正常推出终端

-   别名： `exit`

## `terminal hide[ clear]`

-   功能： 暂时隐藏终端（阻塞），直到输入唤醒指令 `terminal show` 时才重新显示
-   别名： `hide`
-   参数
    -   `clear` ：输入参数 `clear` 则会在隐藏终端之前清空屏幕

## `terminal version`

-   功能： 查询当前的终端版本
-   别名： `-v`
